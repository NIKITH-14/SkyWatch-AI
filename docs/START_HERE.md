## 🎮 How to Use

### Place Equipment
1. Click "PLACE EQUIPMENT" in sidebar
2. Select equipment type & coverage radius
3. Click "ACTIVATE PLACEMENT MODE"
4. Click on map to place systems
5. Watch coverage circles appear

### Run Simulation
1. Click "SIMULATE" in sidebar
2. Adjust speed (1x-10x)
3. Choose scenario
4. Click "START SIMULATION"
5. Watch real-time logs

### Run Tests
1. Click "TEST MODE" in sidebar
2. Choose test type (4 options)
3. View detailed results

### View Analytics
1. Click "ANALYSIS" in sidebar
2. See 4 analytics cards
3. Monitor detailed logs

## 🎨 Customization

### Change Colors
Edit `style.css` lines 10-14:
```css
--accent-green: #00ff41;    /* Primary highlight */
--accent-blue: #00d4ff;     /* Secondary highlight */
--accent-red: #ff0055;      /* Alert color */
```

### Change Map Region
Edit `script.js` lines 66-67:
```javascript
const defaultLat = 20.5937;  /* Change latitude */
const defaultLng = 78.9629;  /* Change longitude */
```

### Add Equipment Types
Edit `script.js` equipmentType dropdown in HTML
