## 🚀 Quick Tips

1. **Login Demo**: Use any username/password - it's permissive for testing
2. **Smooth Controls**: Hold WASD for continuous acceleration
3. **Fast Navigation**: Sidebar buttons provide instant screen switching
4. **Map Interaction**: Click anywhere on the map to reposition current object
5. **AI Monitoring**: Enable for realistic anomaly alerts (simulated)
6. **Speed Control**: Use slider in Settings for 1-10x animation multiplier
7. **Full History**: Movement paths are recorded up to 500 points per object
8. **Dark Theme**: All effects optimized for dark cinematic aesthetics
